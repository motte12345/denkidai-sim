# SPEC.md — 電気代カリキュレーター 仕様書

## サイト概要

- **サイト名**: 電気代カリキュレーター（仮）
- **目的**: 家電ごとの電気代を可視化し、節約や買い替えの判断を支援
- **収益モデル**: AdSense + Amazon/楽天の家電アフィリエイト
- **技術構成**: React静的サイト（Vite） + Cloudflare Pages

## インフラ・運用

- Cloudflare Pages + GitHub自動デプロイ（mainブランチpush）
- Google AdSense / GA4 / Search Console
- 電力料金単価: 年度・電力会社ごとにJSONで管理（年1〜2回更新）

---

## 計算ツール一覧

### 1. 家電別 電気代計算（メイン） — `/calc`

**入力:**
- 家電の消費電力（W）— 直接入力 or プリセット選択
- 1日の使用時間
- 使用日数（毎日 / 週○日 / 月○日）
- 電力料金単価（円/kWh）— デフォルト値あり、手動変更可

**プリセット家電:**
エアコン（冷房/暖房別）、冷蔵庫、洗濯機、乾燥機、テレビ、電子レンジ、ドライヤー、照明、PC、ゲーム機、食洗機、電気ケトル、こたつ、ファンヒーター、空気清浄機、浴室乾燥機 等

- プリセットは一般的なW数をデフォルトで入れつつ、ユーザーが上書き可能

**出力:**
- 1時間 / 1日 / 月 / 年 あたりの電気代
- kWh換算も併記

### 2. 家計の電気代まるごと計算 — `/total`

**入力:** 家電を1つずつ追加（ツール1と同じ入力フォーム、追加ボタンで行を増やす）

**出力:**
- 家電ごとの月額・年額の一覧表
- 合計金額
- 構成比の円グラフ（「エアコンが全体の35%」のように）
- 電気代が高い順のランキング表示

### 3. 買い替え節約シミュレーター — `/replace`

**入力:**
- 現在の家電の消費電力（W）
- 買い替え候補の消費電力（W）
- 1日の使用時間
- 買い替え候補の購入価格

**出力:**
- 年間の電気代削減額
- 購入価格の元が取れるまでの期間
- 5年・10年使った場合のトータルコスト比較（購入価格+電気代）
- 「○年以上使うなら買い替えた方が得」の判定

### 4. エアコンつけっぱなし vs こまめに切る比較 — `/aircon`

**入力:**
- エアコンの消費電力（定格 / 起動時のW数）
- 外出時間
- 外気温と設定温度の差（大まかに選択）

**出力:**
- つけっぱなしの場合の電気代
- 切って再起動した場合の電気代（起動時の電力ピークを加味）
- どちらが安いかの判定と差額
- 「○時間以内の外出ならつけっぱなしが得」の目安表示

**補足:** インバーター制御で単純計算できないため、あくまで目安である旨を明記。

### 5. 電力会社・プラン別 単価比較 — `/plan`

- 主要電力会社の従量電灯プランの料金単価テーブル表示
- 段階料金（〜120kWh / 121〜300kWh / 301kWh〜）を考慮
- **入力:** 月間使用量（kWh）
- **出力:** 各社の月額概算を並べて比較
- 従量電灯（規制料金）に絞る（自由化プランは数が多すぎるため）

### electricityRates.json

```json
{
  "defaultRate": 31,
  "companies": {
    "東京電力": {
      "plan": "従量電灯B",
      "tiers": [
        { "maxKwh": 120, "rate": 29.80 },
        { "maxKwh": 300, "rate": 36.40 },
        { "maxKwhOver": true, "rate": 40.49 }
      ],
      "basicCharge": {
        "30A": 885.72,
        "40A": 1180.96,
        "50A": 1476.20,
        "60A": 1771.44
      }
    }
  },
  "lastUpdated": "2026-04-01"

```

### appliances.json

```json
{
  "エアコン（冷房・6畳）": { "watt": 500, "category": "空調" },
  "冷蔵庫（400L）": { "watt": 40, "category": "キッチン", "alwaysOn": true },
  "ドラム式洗濯乾燥機": { "watt": 1200, "category": "洗濯" },

```

- `alwaysOn: true` の家電は使用時間を24時間に固定してデフォルト表示

## 差別化ポイント

1. **家電プリセット** — W数を調べなくてもすぐ使える（既存サイトはW数直接入力が多い）
2. **複数家電の構成比グラフ** — 「何が電気代を食っているか」が一目でわかる
3. **買い替え損益分岐** — 「この冷蔵庫に買い替えれば3年で元が取れます」→ アフィリエイトへ自然に接続
4. **エアコンつけっぱなし問題** — 検索ボリュームが大きく、単独で流入が見込める

---

## 免責事項

- 概算であり、実際の電気代は使用環境・契約内容により異なる
- 電力料金は改定される場合がある。最新の料金は各電力会社のサイトで確認
- エアコンの比較は簡易モデルであり、実際のインバーター制御とは異なる
- 特定の電力会社や家電製品を推奨するものではない
